# Annotation Garden Management

Central coordination and documentation repository for the Annotation Garden Initiative.

## Purpose

This repository serves as the organizational hub for the Annotation Garden Initiative, containing:
- Project documentation and white papers
- Organizational guidelines and policies
- Cross-repository coordination
- Meeting notes and decisions
- Implementation roadmaps

## Key Documents

- `agi-white-paper.md`: Comprehensive project vision and technical architecture
- `CONTRIBUTING.md`: Guidelines for contributing to AGI repositories
- `GOVERNANCE.md`: Decision-making processes and organizational structure

## Related Repositories

- [assets](https://github.com/Annotation-Garden/assets): Logo, design guidelines, templates
- [website](https://github.com/Annotation-Garden/annotation-garden.github.io): Main AGI website
- [image-annotation](https://github.com/Annotation-Garden/image-annotation): Web-based annotation tool

## Standards and Integration

AGI builds on established neuroscience standards:
- **BIDS**: Brain Imaging Data Structure, particularly [stimuli-BIDS (BEP042)](https://github.com/bids-standard/bids-specification/pull/2022)
- **HED**: Hierarchical Event Descriptors ([hedtags.org](https://hedtags.org))
